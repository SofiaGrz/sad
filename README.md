# 📊 Clasificador de Datos con Machine Learning

Este proyecto implementa un sistema de clasificación de datos en Python utilizando diferentes algoritmos de Machine Learning como:

- k-Nearest Neighbors (kNN)
- Árboles de Decisión
- Random Forest

Incluye un pipeline completo de:
- Preprocesamiento de datos
- Entrenamiento
- Evaluación
- Predicción

---

# ⚙️ Requisitos

- Crear entorno virtual
  - python -m venv venv
  - source venv/bin/activate
- pip install -r requirements.txt
